# Exia

AI agent experiments in diagnosing CI/CD failures.

All the scripts here support https://github.com/do-exploit/exia-examples .

> [!CAUTION]
> This implementation is not best practice. You should consider security and simple infrastructure provisioning or deployment. This is only for exploration and focuses on functionality, not production readiness.

## Infrastructure Requirements

1. [Kubernetes Cluster](https://microk8s.io/docs/install-alternatives)
2. [n8n](https://docs.n8n.io/hosting/installation/docker/)
